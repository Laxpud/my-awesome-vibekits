# Agent Skills

个人技能（skills）集合，用于管理和分享各种实用工具和功能。

## 目录结构

```
├── skills/                # 技能目录
│   └── pyproject-standard/ # 标准pyproject.toml配置技能
│       └── SKILL.md       # 技能说明文件
├── .gitignore             # Git忽略文件
└── README.md              # 仓库说明文件
```

## 如何使用

1. 克隆仓库到本地
2. 浏览 `skills` 目录查看可用技能
3. 按照各技能的说明文件使用

## 贡献

欢迎添加新的技能或改进现有技能，提交Pull Request即可。
